# p5.wrapper
A lightweight declarative wrapper for p5.js that lets you build interactive sketches using Web Components or frameworks like Zikojs, React, VanJS...

## API 
### Scenes
 - P5Canvas2D(...shapes)
 - P5Canvas3D(...shapes)
### Shapes 
 - P5Circle 
 - P5Rect 
 - P5Ellipse
